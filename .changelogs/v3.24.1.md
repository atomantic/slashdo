# Release v3.24.1

Released: 2026-07-23

## Code review
- **Review-only reviewers can no longer quietly rewrite your working tree** — a `--review-with` pass is supposed to report findings and let the orchestrator apply them, but every reviewer was launched with full write access and the "don't touch the tree" contract existed only as a line in the prompt. A reviewer that decided to fix things itself would silently overwrite uncommitted work, and you'd only notice when your edits reappeared undone. Codex's review pass now runs in an OS-enforced read-only sandbox (verified: it still reads the diff, commit graph and base tree and returns normal findings, while writes fail outright). The CLIs with no read-only mode — `claude`, `agy`, `grok` — are now backstopped instead: the loop snapshots HEAD, the index, tracked-file contents and untracked files before the review, and if the reviewer modified anything it restores that state wholesale and warns, keeping the findings. Reviewer-applies mode (`--reviewer-applies`) is unaffected — writing is the point there.

## Dependencies
- Bump `actions/setup-node` from 6 to 7 in the GitHub Actions workflows.
