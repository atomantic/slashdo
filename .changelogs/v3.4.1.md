# Release v3.4.1

Released: 2026-05-31

## Bug Fixes

- **Fix crash on `npx slash-do@latest`** when the legacy Gemini CLI path (`~/.gemini/commands/do/`) exists on disk. `detectInstalled()` was returning `gemini-legacy` as an install target, causing `ENVIRONMENTS['gemini-legacy']` to be undefined and crashing with `Cannot read properties of undefined (reading 'name')`. Legacy environments are now only included during `--uninstall` runs, not during auto-detection for install.
