# Release v2.14.3

Released: 2026-05-16

## Fixed

- **do:fpr**: Resolve fork via origin remote — `fpr` now correctly detects the upstream from the `origin` remote URL rather than relying on a hardcoded assumption, fixing cases where the fork's origin pointed to the user's fork instead of the upstream repo (#36)
- **do:fpr**: Use portable `sed` pipeline for origin slug extraction — replaces bash-specific parameter expansion with a POSIX-compatible sed chain so the command works on macOS/BSD as well as Linux
- **do:fpr**: Handle `.git/` suffix and guard non-GitHub origins — strips trailing `.git` from origin URLs and emits a clear error (including the raw origin URL) when the remote is not a GitHub host
- **do:fpr**: Add POSIX guard and include raw origin URL in error — improves diagnostics when origin detection fails
