# Release v3.24.0

Released: 2026-07-12

## Task planning
- **[issue-111] Enhance drafted issues before filing** — `/do:plan-task` gains a `--enhance-with <list>` flag that routes the drafted issue through a sequential pipeline of enhancement agents (`codex`, `claude`, `agy`, `grok`) before the approval gate, each sharpening the previous one's draft — a cheap second/third opinion for a more decision-complete issue. Grok is now a first-class enhancement agent.

## Code review
- **[issue-120] Grok as a code reviewer** — `grok` is now a first-class `--review-with` reviewer everywhere reviewers run (`/do:pr`, `/do:rpr`, `/do:review`, `/do:better`, `/do:better-swift`, `/do:depfree`, `/do:release`), with per-reviewer model pinning via `grok[<model>]` and saveable defaults through `/do:config`. It reviews your working tree locally, headlessly, before the PR is pushed.
