## Changed

- **Statusline update notifications are now generic.** The statusline previously hardcoded two checks (`gsd-update-check.json` → `⬆ /gsd:update` and `slashdo-update-check.json` → `⬆ /do:update`). It now scans the cache directory for every `*-update-check.json` file and renders a badge from each file's own `command` field, so any tool that drops an update-check cache file gets a notification without the statusline knowing the tool's name. The slashdo update-check hook now writes `command: '/do:update'` so the new scanner picks it up.
