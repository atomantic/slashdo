# Release v3.4.2

Released: 2026-05-30

## Bug Fixes

- **Antigravity (`agy`) now lists all installed skills instead of collapsing them into one.** The Agent Skills standard used by `agy` and Codex requires a `name` field in each `SKILL.md` frontmatter that matches the skill directory. slashdo's source commands only defined `description`, so every installed `SKILL.md` lacked `name` and `agy` couldn't disambiguate the 16 skills — its UI showed only a single entry. The transformer now injects `name` (derived from the skill directory via a shared `getSkillName` helper, ordered ahead of `description` per the standard) for directory-namespacing environments only. An existing `name` is left untouched, and Claude (subdirectory) and OpenCode (flat) installs are unaffected.
