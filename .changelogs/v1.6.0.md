# Release v1.6.0

Released: 2026-03-07

## Added

- Test enhancement phase (Phase 4c) in `do:better` — evaluates and improves test quality, not just coverage
- Agent 7 upgraded to detect vacuous tests (tests that don't test anything), weak test patterns, and coverage gaps
- Dedicated test agent that writes/fixes tests with falsifiability verification (tests must fail when code is broken)
- Test files now get their own `better/tests` PR branch in the per-category PR pipeline

## Improved

- Code review checklist: broadened items for UI action/state consistency, internal identifier renames, route/URL backward compatibility, and data model lifecycle changes
- Review process (`do:review`): added deep checks for status/enum lifecycle changes and concept rename tracing
- Template syntax standardized to consistent placeholder format across `do:better`
- `improve:review` skill enforces no-project-references rule
