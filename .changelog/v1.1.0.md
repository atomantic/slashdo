# Release v1.1.0

Released: 2026-03-02

## Added

- Interactive model profile selection for `do:better` — users choose quality/balanced/budget before the pipeline runs
- Model profile table mapping audit agents (Explore) and remediation agents (general-purpose) to opus/sonnet/haiku per tier
- Rationale documentation for each model assignment in the command itself

## Changed

- `do:better` Phase 1 audit agents now accept a configurable `model` parameter instead of always inheriting the session model
- `do:better` Phase 3 remediation agents now accept a configurable `model` parameter
- Balanced profile (default) uses Sonnet for both audit and remediation, saving significant cost vs all-Opus
- Budget profile uses Haiku for audit (7 agents) while keeping Sonnet for code-writing remediation agents
